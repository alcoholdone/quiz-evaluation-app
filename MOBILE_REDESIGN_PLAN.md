# แผน Mobile Redesign (โทรศัพท์ < 640px)

> สถานะ: **กำลังทำบน branch `mobile-redesign`**
> ความคืบหน้า: ✅ item 1, 3, 4, 6, 8 ทำแล้ว · ✅ item 2, 7 เดิมรองรับอยู่แล้ว ·
> ⏳ item 5 (ตาราง→การ์ด) ค้าง — ตัวเดียวที่ต้องแตะ JS (เพิ่ม `data-label` ตอน render)
> ลำดับความสำคัญจริง: **iPad + Desktop เป็นหลัก**, mobile เป็นผลพลอยได้

## หลักการ
บนมือถือไม่ใช่ "ย่อ desktop" แต่ออกแบบใหม่เฉพาะจุดให้เหมาะกับนิ้ว/จอแคบ:
**เปลี่ยนรูปร่าง · หุบ · ย่อ · ซ่อน/โชว์** ตามบริบท โดย**ไม่แตะ JS / element IDs**
(ใช้ CSS media query `<640px` + class toggle เท่านั้น เพื่อไม่ให้กระทบเกม/sync)

## รายการงาน (เรียงตาม impact)

### 1. Navigation — แท็บโหมด → Bottom Tab Bar  ★ impact สูงสุด
- ปัจจุบัน: ปุ่มแท็บ 4 อันเรียง wrap ด้านบน กินพื้นที่ + เอื้อมยาก
- มือถือ: ย้ายเป็น **แถบล่างติดจอ (fixed bottom)** แบบไอคอน + label สั้น
  (📖 ฝึก · 🎮 เกม · 📝 เขียน · 📊 ผล) — เอื้อมด้วยนิ้วโป้งง่าย เหมือนแอปจริง
- เผื่อพื้นที่ล่างด้วย `padding-bottom` + `env(safe-area-inset-bottom)`
- **ความเสี่ยง:** ต่ำ (toggle ด้วย CSS, ปุ่มเดิมยังเป็นปุ่มเดิม)

### 2. ปุ่มโหมด/ตัวเลือก (mode selectors, quiz length)
- เปลี่ยน pill ใหญ่ → **เต็มความกว้าง stack แนวตั้ง** หรือ grid 2 คอลัมน์
- ลด padding, ดันไอคอนนำหน้า, ตัดข้อความช่วยเหลือที่ยาว

### 3. Card — ปรับรูปร่างบนมือถือ
- ลด padding ด้วย `--space-panel`
- การ์ดเต็มขอบจอ (edge-to-edge): ตัด margin ข้าง, ลด border-radius, ลดเงา
- ข้อความช่วยเหลือยาว → ซ่อนใต้ปุ่ม info (ⓘ) แบบกดเปิด

### 4. Modal → Bottom Sheet
- modal กลางจอ → **เลื่อนขึ้นจากด้านล่าง (bottom sheet)** เต็มความกว้าง
- รองรับปัดลงเพื่อปิด (swipe-to-dismiss) + ปุ่มปิดชัดเจน

### 5. Dashboard (รายงานผล) — ยากสุด
- กราฟ: stack 1 คอลัมน์ + จำกัดความสูง (ไม่ให้สูงล้นจอ)
- **ตาราง → การ์ดรายการ** (label:value ซ้อนกัน) แทน scroll แนวนอน อ่านง่ายกว่ามาก
- stats overview: grid 2 คอลัมน์แบบกระชับ

### 6. Header — หุบบนมือถือ
- ลดขนาด title, ย่อป้ายดาวเป็น chip เล็ก
- พิจารณา sticky header แบบกระชับเมื่อ scroll

### 7. องค์ประกอบเกม (letter tiles / options)
- การ์ดตัวอักษร/ตัวเลือก: 1 คอลัมน์, tap target ใหญ่ขึ้น, ระยะห่างนิ้ว

### 8. ซ่อน/ลดสิ่งรบกวน
- ซ่อน background blobs ตกแต่งบนมือถือ (เพิ่ม perf + สะอาดตา)
- ซ่อน subtitle/ข้อความเสริมที่ไม่จำเป็น

- ทุกอย่างทำผ่าน CSS `@media (max-width: 640px)` + class — **ห้ามแตะ JS/IDs** ที่เกม/sync ใช้
- ต้องทดสอบจริงบนมือถือ (โดยเฉพาะ bottom sheet + ตาราง→การ์ด) เพราะ verify ด้วยตาบนจอจริงเท่านั้น
